- status: approved

- 项目代号：WEATHER-APP

## 项目概述

一个前后端分离的天气查询 Web 应用。用户在浏览器输入城市名称，系统调用和风天气（QWeather）API 返回实时天气信息，包括温度、天气描述、风向等。UI 简洁清晰，代码结构清晰可运行。

## 功能列表

| 编号 | 功能 | 优先级 | 描述 |
|------|------|--------|------|
| F1 | 城市搜索 | P0 | 输入城市名称，调用 QWeather 城市查询接口获取城市 ID |
| F2 | 实时天气查询 | P0 | 根据城市 ID 调用 QWeather 实时天气接口，返回天气数据 |
| F3 | 天气信息展示 | P0 | 在页面上展示温度、天气描述、风向等信息 |
| F4 | 简洁 UI | P0 | 界面简洁清晰，输入框 + 查询按钮 + 天气卡片展示 |

## 验收标准

1. **F1**：输入"北京"或"beijing"能返回正确的城市列表
2. **F2**：选择城市后能获取到实时天气数据（温度、天气状况、风向）
3. **F3**：页面清晰展示温度（°C）、天气描述文字、风向风力信息
4. **F4**：页面在主流浏览器（Chrome/Safari/Firefox）中正常显示，布局整洁

## 范围边界

### 做什么

- 前后端分离架构
- 城市名称搜索（模糊匹配）
- 实时天气查询与展示（温度、天气描述、风向）
- 完整可运行代码 + 运行步骤说明

### 不做什么

- 不做天气预报（仅实时天气）
- 不做用户登录/注册
- 不做数据持久化/历史记录
- 不做移动端适配（桌面浏览器即可）
- 不做部署上线（本地运行即可）
